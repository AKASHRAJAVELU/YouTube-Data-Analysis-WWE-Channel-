# Text-Data-Analysis-WWE-YOUTUBE-Channel-
This project analyzes YouTube video data and comments from the WWE channel.  The goal is to understand viewer engagement, popular categories, and audience reactions.  Python data analytics libraries are used to process and visualize the data.

 # Objectives

To analyze YouTube video performance. 

To measure views, likes, and engagement rate.

To identify most popular video categories.

To analyze user comments and emojis.

To visualize insights using charts.

# Tools and Technologies Used

Python

Jupyter Notebook

Pandas – data processing

NumPy – numerical operations

Matplotlib / Plotly – data visualization

YouTube Dataset (CSV files)

# Data Collection

The dataset contains YouTube video statistics and comments.

Data includes:

Video ID

Category Name

Views

Likes

Comment Count

Engagement Rate

# Data Preprocessing

Dataset Link: https://drive.google.com/drive/folders/1sWXB6SC4adLrBUorMwYwq1jSkWC9GXsp?usp=drive_link

Steps performed before analysis:

Merging multiple CSV files into one dataset.

Removing invalid or corrupted rows.

Handling missing values.

Formatting columns properly.


# Data Analysis Techniques

The following techniques were applied:

GroupBy analysis to summarize video performance.

Engagement rate calculation.

Emoji extraction from comments.

Sentiment/engagement analysis.

# Data Visualization

Several visualizations were created to understand the data:

Bar Chart – most used emojis in comments.

Treemap Chart – category attention share.

Engagement efficiency visualization.

Views and engagement comparison charts.

# Key Insights

Certain video categories receive higher engagement.

Some emojis are frequently used in audience reactions.

High view count does not always mean high engagement rate.

# Conclusion

Data analytics helps understand viewer behavior on YouTube.

Visualizations make it easier to identify content performance trends.
This project demonstrates how Python can be used for social media data analysis.
